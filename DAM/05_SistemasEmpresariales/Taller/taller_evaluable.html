Taller Evaluable - Sistemas Empresariales

Enunciado: Implementa la funcion calcularFactura que reciba un array de lineas de pedido, cada una con la estructura { producto: string, cantidad: number, precioUnitario: number, categoria: string }. Debe devolver un objeto con:

  • subtotal: suma de cantidad * precioUnitario de todas las lineas
  • descuento: si el subtotal supera 100 EUR, aplicar 10% de descuento; si supera 500, 20%
  • iva: 21% sobre (subtotal - descuento)
  • total: subtotal - descuento + iva
  • lineas: numero de lineas procesadas
