Taller Evaluable - Servicios y Procesos

Enunciado: Implementa la funcion asincrona simularPeticion que reciba un array de URLs (strings simuladas) y las procese secuencialmente. Para cada URL, debe simular una peticion que tarda un tiempo aleatorio entre 100 y 500 ms y, si la URL contiene la palabra "error", debe lanzar una excepcion con el mensaje "Error en: URL". La funcion debe devolver un array con los resultados exitosos y los errores capturados, con formato { resultados: [...], errores: [...] }.

Pista: usa new Promise y async/await.
