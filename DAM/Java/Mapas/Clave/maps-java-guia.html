01

Métodos básicos — HashMap

Crear y añadir

put / get
Map<Integer, String> mapa = new HashMap<>();

// put — añade o reemplaza
mapa.put(1, "Ana");
mapa.put(2, "Carlos");
mapa.put(3, "Beatriz");
mapa.put(1, "Andrea");  // reemplaza clave 1

// get — obtener por clave
mapa.get(2);          // → "Carlos"
mapa.get(99);         // → null

// size / isEmpty
mapa.size();          // → 3
mapa.isEmpty();       // → false

Verificar y eliminar

contains / remove
// containsKey / containsValue
mapa.containsKey(3);         // → true
mapa.containsValue("Ana");  // → true

// remove — elimina por clave
mapa.remove(2);

// clear — vacía todo
Map<Integer, String> temp =
    new HashMap<>(mapa);
temp.clear();
temp.isEmpty();  // → true

Vistas del mapa

keySet / values / entrySet
// keySet() — conjunto de claves
mapa.keySet();     // → {1, 3, 4, 5}

// values() — colección de valores
mapa.values();     // → [Andrea, Beatriz, David…]

// entrySet() — recorrer pares
for (Map.Entry<Integer, String> e
        : mapa.entrySet()) {
    System.out.println(
        e.getKey() + " → " + e.getValue()
    );
}
02

Métodos modernos — Java 8+

putIfAbsent / getOrDefault

Java 8+
// putIfAbsent — solo añade si NO existe
mapa.putIfAbsent(6, "Fernando");
mapa.putIfAbsent(1, "NUEVO"); // ignorado, ya existe
mapa.get(1);  // sigue siendo "Andrea"

// getOrDefault — valor por defecto si no existe
mapa.getOrDefault(7, "Desconocido"); // → "Desconocido"
mapa.getOrDefault(3, "Desconocido"); // → "Beatriz"

replace / forEach

Java 8+
// replace — solo si la clave existe
mapa.replace(3, "Beatriz_Mod");

// replace con verificación de valor antiguo
boolean ok = mapa.replace(3,
    "Beatriz_Mod", "Beatriz_Final"
); // → true si coincide valor anterior

// forEach — iteración funcional
mapa.forEach((clave, valor) ->
    System.out.println("[" + clave + "] = " + valor)
);

compute / merge

Java 8+ avanzado
Map<String, Integer> cnt = new HashMap<>();

// computeIfAbsent — calcula si no existe
cnt.computeIfAbsent("manzana", k -> 0);

// compute — actualiza con función
cnt.compute("manzana",
    (k, v) -> (v == null) ? 1 : v + 1
);

// merge — combina valores
cnt.merge("naranja", 1, Integer::sum); // 1ª vez → 1
cnt.merge("naranja", 2, Integer::sum); // 2ª vez → 3
03

Tipos de Map y mapas inmutables

HashMap / LinkedHashMap / TreeMap

tipos
// HashMap — sin orden garantizado
Map<Integer, String> hm = new HashMap<>();

// LinkedHashMap — orden de inserción
Map<Integer, String> lhm = new LinkedHashMap<>();
lhm.put(3,"Tres"); lhm.put(1,"Uno"); lhm.put(2,"Dos");
// → {3=Tres, 1=Uno, 2=Dos}

// TreeMap — orden natural de claves
Map<Integer, String> tm = new TreeMap<>();
// → {1=Uno, 2=Dos, 3=Tres}

// TreeMap con orden inverso
Map<Integer, String> tmd =
    new TreeMap<>(Collections.reverseOrder());
// → {3=Tres, 2=Dos, 1=Uno}

Mapas inmutables — Java 9+

Map.of
// Map.of — hasta 10 pares
Map<String, Integer> fijo = Map.of(
    "Ana", 25,
    "Carlos", 30,
    "Beatriz", 28
);

// Map.ofEntries — más de 10 pares
Map<String, Integer> grande = Map.ofEntries(
    Map.entry("Ana", 25),
    Map.entry("Carlos", 30)
    // más entradas...
);

// Convertir a mutable
Map<String, Integer> mutable =
    new HashMap<>(fijo);
mutable.put("Elena", 22); // ✅

Resumen de métodos

Todos los métodos

referencia completa
put(k, v)Añade o reemplaza par clave-valor
get(k)Obtiene valor por clave
remove(k)Elimina entrada por clave
containsKey(k)¿Existe la clave?
containsValue(v)¿Existe el valor?
size()Número de entradas
isEmpty()¿Está vacío?
clear()Elimina todas las entradas
keySet()Conjunto de claves
values()Colección de valores
entrySet()Conjunto de pares clave-valor
putIfAbsent(k, v)Añade solo si no existe (Java 8+)
getOrDefault(k, def)Valor o por defecto (Java 8+)
replace(k, v)Reemplaza si existe (Java 8+)
replace(k, old, new)Reemplaza con verificación
forEach((k,v)->…)Iteración funcional (Java 8+)
computeIfAbsent(k, f)Calcula valor si no existe
compute(k, bif)Actualiza valor con función
merge(k, v, bif)Combina valores
Map.of(…)Mapa inmutable hasta 10 pares (Java 9+)
Map.ofEntries(…)Mapa inmutable sin límite de pares
Map.entry(k, v)Crea entrada para ofEntries
