Tipos de Set

01

HashSet — sin orden

máximo rendimiento
Set<String> hashSet = new HashSet<>();
hashSet.add("Manzana");
hashSet.add("Banana");
hashSet.add("Naranja");
hashSet.add("Manzana"); // duplicado ignorado

hashSet.size();                    // → 3
hashSet.contains("Banana");       // → true
hashSet.isEmpty();                 // → false
⚠ OrdenEl HashSet no garantiza el orden de los elementos. Usar solo cuando el orden no importa.

LinkedHashSet — orden inserción

mantiene orden
Set<String> lhs = new LinkedHashSet<>();
lhs.add("Z");
lhs.add("A");
lhs.add("M");
lhs.add("A"); // duplicado ignorado
// → [Z, A, M]  (orden de inserción)

// Caso práctico: eliminar duplicados
// manteniendo la primera aparición
List<String> lista = Arrays.asList(
    "Ana","Carlos","Ana","Beatriz"
);
Set<String> sinDup = new LinkedHashSet<>(lista);
// → [Ana, Carlos, Beatriz]

TreeSet — orden automático

orden natural
Set<String> ts = new TreeSet<>();
ts.add("Zanahoria");
ts.add("Manzana");
ts.add("Banana");
// → [Banana, Manzana, Zanahoria] ← alfabético

TreeSet<String> cast = (TreeSet<String>) ts;
cast.first();             // → "Banana"
cast.last();              // → "Zanahoria"
cast.headSet("N");        // elementos < "N"
cast.tailSet("M");        // elementos >= "M"

// Comparator personalizado (por longitud)
Set<String> porLong = new TreeSet<>(
    Comparator.comparingInt(String::length)
);
porLong.add("A"); porLong.add("Elefante"); porLong.add("Gato");
// → [A, Gato, Elefante]

Set<Integer> A = new HashSet<>(Arrays.asList(1,2,3,4,5));
Set<Integer> B = new HashSet<>(Arrays.asList(4,5,6,7,8));

Set<Integer> union = new HashSet<>(A); union.addAll(B);
// A ∪ B → {1,2,3,4,5,6,7,8}

Set<Integer> inter = new HashSet<>(A); inter.retainAll(B);
// A ∩ B → {4,5}

Set<Integer> dif = new HashSet<>(A); dif.removeAll(B);
// A − B → {1,2,3}

Set<Integer> sim = new HashSet<>(A); sim.addAll(B);
Set<Integer> tmp = new HashSet<>(A); tmp.retainAll(B);
sim.removeAll(tmp);
// A △ B → {1,2,3,6,7,8}

Set<Integer> sub = new HashSet<>(Arrays.asList(2,3));
A.containsAll(sub); // A ⊇ {2,3} → true

Métodos de instancia y conjuntos inmutables

03

Eliminar y modificar

remove / clear
Set<String> frutas = new HashSet<>(
    Arrays.asList("Manzana","Banana","Naranja")
);

// remove — elimina elemento específico
frutas.remove("Banana");

// removeAll — elimina múltiples
frutas.removeAll(Arrays.asList("Manzana","Pera"));
// "Pera" no existe → se ignora

// retainAll — mantiene solo los indicados
Set<String> mantener = Set.of("Naranja","Uva");
frutas.retainAll(mantener);

// clear — vacía el conjunto
frutas.clear();
frutas.isEmpty(); // → true

Conjuntos inmutables + iteración

Java 9+ / Java 8+
// Set.of — inmutable (Java 9+)
Set<String> inm = Set.of("Rojo", "Verde", "Azul");
// inm.add("X") → UnsupportedOperationException ❌

// Convertir a mutable
Set<String> mut = new HashSet<>(inm);
mut.add("Amarillo"); // ✅

// Iteración for-each
for (String c : mut) System.out.print(c + " ");

// forEach con lambda (Java 8+)
mut.forEach(c -> System.out.print(c + " "));

// Stream — filtrar y operar
mut.stream().filter(c -> c.length() > 4)
           .forEach(System.out::println);
