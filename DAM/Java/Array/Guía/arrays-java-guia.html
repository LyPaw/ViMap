I

Arrays Unidimensionales (Vectores)

Colección de tamaño fijo · mismo tipo · índice desde 0

Declaración y Creación

paso 1
// Solo referencia (pila)
double[] numeros;

// Reserva memoria (montículo)
numeros = new double[10];

// Forma concisa (ambos juntos)
double[] numeros = new double[10];
⚠ Nota La declaración crea solo la referencia en la pila. El new reserva el espacio en el montículo.

Inicialización

paso 2
// Por índice (relleno manual)
double[] miLista = new double[3];
miLista[0] = 1.2;
miLista[1] = 2.2;
miLista[2] = 3.2;

// Directa (declaración + valores)
double[] miLista = {1.2, 1.3, 1.4};

// Otra forma válida
int[] valores = new int[] {1, 2, 4};

Recorrido

paso 3
// For estándar (con índice)
for (int i = 0; i < miLista.length; i++) {
  System.out.println(miLista[i]);
}

// For-each (sin índice, más limpio)
for (double u : miLista) {
  System.out.println(u);
}

// Impresión rápida (depuración)
System.out.println(
  Arrays.toString(miLista)
);
II

Arrays Bidimensionales (Matrices)

Tabla de filas y columnas · doble índice [fila][columna]

Declaración y Creación

paso 1
// 8 filas × 8 columnas
int[][] distancia = new int[8][8];

// Longitud de un array [3][4]:
x.length        // → 3 (filas)
x[0].length     // → 4 (columnas)

Inicialización

paso 2
// Por índice
distancia[0][0] = 0;
distancia[0][1] = 615;

// Directa (llaves anidadas)
int[][] array = {
  {1, 2, 3},    // Fila 0
  {4, 5, 6},    // Fila 1
  {7, 8, 9}     // Fila 2
};

Recorrido con bucles anidados

paso 3
for (int fila = 0; fila < matriz.length; fila++) {
  for (int col = 0;
       col < matriz[fila].length; col++) {
    System.out.print(
      matriz[fila][col] + " "
    );
  }
  // Salto de línea por fila
  System.out.println();
}
III

ArrayList — Colecciones Dinámicas

import java.util.ArrayList · crece y se reduce · solo objetos

Declaración

creación
// Sin genéricos (genera warning ⚠)
ArrayList lista = new ArrayList();

// Con genéricos ✅ (recomendado)
ArrayList<String> lista =
    new ArrayList<>();

// Usando interfaz padre (mejor)
List<String> lista =
    new ArrayList<>();
💡 Genéricos Aseguran que solo se agreguen datos del tipo especificado y eliminan la necesidad de castings.

Métodos principales

operaciones
add(elem)Añade al final de la lista
add(i, elem)Inserta en el índice i
get(i)Devuelve el elemento del índice i
size()Número de elementos actuales
remove(i)Elimina el elemento en índice i
contains(o)¿Existe ese elemento? → boolean
toString()Imprime toda la lista como texto
clear()Vacía la lista completamente

Relleno y Recorrido

uso
ArrayList<String> cities =
    new ArrayList<>();

cities.add("Londres");
cities.add("Madrid");
cities.add(2, "Roma"); // inserta en [2]

// For normal (con índice)
for (int i = 0; i < cities.size(); i++) {
  System.out.println(cities.get(i));
}
// For-each
for (String c : cities) {
  System.out.println(c);
}
// Impresión rápida
System.out.println(cities.toString());

Ejercicios Resueltos

Arrays unidimensionales y bidimensionales

Ejercicio 1 · Suma y Media

1D
Array double · for estándar
Crear array de precios y calcular suma y media.
double[] precios = new double[5];
precios[0] = 15.50; precios[1] = 20.00;
precios[2] =  5.25; precios[3] = 12.00;
precios[4] =  8.75;
double suma = 0;
for (int i = 0; i < precios.length; i++) {
  suma += precios[i];
}
double media = suma / precios.length;
System.out.println("Suma: "  + suma);
System.out.println("Media: " + media);
Resultado esperado
Suma total: 61.5
Valor medio: 12.3

Ejercicio 2 · Conteo

1D
Array int · for-each · condicional
Contar calificaciones ≥ 90 con for-each.
int[] cal = {85,92,78,95,88,91,75,99};
int sobres = 0;
for (int c : cal) {
  if (c >= 90) sobres++;
}
System.out.println(
  "Hay " + sobres + " sobresalientes"
);
Resultado esperado
Total: 8
Hay 4 alumnos con sobresaliente

Ejercicio 3 · Arrays.toString()

1D
Array String · utilidad Arrays
Imprimir array completo en una sola línea.
import java.util.Arrays;

String[] dias = {
  "Lunes", "Martes", "Miércoles",
  "Jueves", "Viernes",
  "Sábado", "Domingo"
};
System.out.println(
  Arrays.toString(dias)
);
Resultado esperado
[Lunes, Martes, Miércoles,
 Jueves, Viernes, Sábado, Domingo]

Ejercicio 4 · Matriz Identidad

2D
Matriz 3×3 · acceso por índice
Declarar matriz identidad y acceder a posición central.
int[][] m = {
  {1, 0, 0},
  {0, 1, 0},
  {0, 0, 1}
};
System.out.println(m[1][1]);     // 1
System.out.println(m.length);    // 3 filas
System.out.println(m[0].length); // 3 cols
Resultado esperado
Valor [1][1]: 1
Filas: 3 · Columnas: 3

Ejercicio 5 · Suma por Fila

2D
Matriz 4×5 · bucles anidados
Calcular suma de elementos por cada fila.
int[][] datos = {
  {10,2,5,8,1},
  {3,15,7,6,9},
  {1,2,3,4,5},
  {10,10,10,10,10}
};
for (int f=0; f<datos.length; f++) {
  int suma = 0;
  for (int c=0; c<datos[f].length; c++) {
    suma += datos[f][c];
  }
  System.out.println(
    "Fila "+f+": "+suma
  );
}
Resultado esperado
Fila 0: 26  · Fila 1: 40
Fila 2: 15  · Fila 3: 50
