Fechas en Java

import java.time.* · Java 8+ · Guía de referencia rápida

java.time
01 — Clases principales

Clases de java.time

LocalDate
Fecha sin hora (año, mes, día). Ej: 2024-07-06
LocalTime
Hora sin fecha (hora, min, seg). Ej: 08:40:15
LocalDateTime
Fecha + hora sin zona horaria. Combina las dos anteriores.
ZonedDateTime
Como LocalDateTime pero con zona horaria concreta.
Instant
Timestamp UNIX en nanosegundos desde epoch (1/1/1970 UTC).
Period
Diferencia entre fechas en días, meses, años.
Duration
Diferencia entre horas en horas, minutos, segundos, nanos.
ChronoUnit
Enumeración de unidades de tiempo para between().

03 — Métodos factoría (crear objetos)

now() — fecha/hora actual

LocalDate     ld  = LocalDate.now();
LocalTime     lt  = LocalTime.now();
LocalDateTime ldt = LocalDateTime.now();
ZonedDateTime zdt = ZonedDateTime.now();
Instant       ins = Instant.now();

// Salida ejemplo (ISO 8601):
// 2024-07-06
// 08:40:15.123
// 2024-07-06T08:40:15.123

of() — fecha/hora concreta

LocalDate fecha = LocalDate.of(2024, Month.MAY, 15);
// También: .of(2024, 5, 15)

LocalTime hora = LocalTime.of(8, 40, 15);

LocalDateTime ldt = LocalDateTime.of(
    2024, Month.MAY, 15, 8, 40, 15
);

// ⚠️ Fecha inválida → lanza excepción:
// LocalDate.of(2019, 2, 29)  ← ERROR
04 — Extraer partes de una fecha/hora

Métodos getXXX()

getYear()Devuelve el año como int
getMonth()Devuelve enum Month (JANUARY…)
getMonthValue()Devuelve el mes como int (1–12)
getDayOfMonth()Día del mes (1–31)
getDayOfWeek()Enum DayOfWeek (MONDAY…)
getDayOfYear()Día del año (1–365/366)
getHour()Hora (0–23)
getMinute()Minutos (0–59)
getSecond()Segundos (0–59)
getNano()Nanosegundos

Ejemplo getters

LocalDateTime ldt = LocalDateTime.now();

int    anio  = ldt.getYear();
Month  mes   = ldt.getMonth();
int    dia   = ldt.getDayOfMonth();
DayOfWeek dow = ldt.getDayOfWeek();
int    hora  = ldt.getHour();
int    min   = ldt.getMinute();

System.out.println(dow); // MONDAY
System.out.println(mes); // JULY
05 — Sumar y restar tiempo

plus / minus — LocalDate

plusDays(n)Suma n días
minusDays(n)Resta n días
plusWeeks(n)Suma n semanas
minusWeeks(n)Resta n semanas
plusMonths(n)Suma n meses
minusMonths(n)Resta n meses
plusYears(n)Suma n años
minusYears(n)Resta n años

plus / minus — LocalTime

plusHours(n)Suma n horas
minusHours(n)Resta n horas
plusMinutes(n)Suma n minutos
minusMinutes(n)Resta n minutos
plusSeconds(n)Suma n segundos
minusSeconds(n)Resta n segundos
plusNanos(n)Suma n nanosegundos
minusNanos(n)Resta n nanosegundos

Ejemplo suma/resta

LocalDate hoy    = LocalDate.now();
LocalDate manana = hoy.plusDays(1);
LocalDate ayer   = hoy.minusDays(1);
LocalDate mesQ   = hoy.plusMonths(3);

LocalTime hora   = LocalTime.of(8, 0);
LocalTime tarde  = hora.plusHours(6);  // 14:00

// ⚠️ Son inmutables → siempre nueva instancia
// plusXXX(-n) == minusXXX(n)
06 — Ajustadores temporales (TemporalAdjusters)

Ajustadores disponibles

firstDayOfMonth()Primer día del mes actual
lastDayOfMonth()Último día del mes actual
firstDayOfNextMonth()Primer día del mes siguiente
firstDayOfNextYear()Primer día del año siguiente
firstDayOfYear()Primer día del año actual
lastDayOfYear()Último día del año actual
next(DayOfWeek)Próximo día de la semana indicado
previous(DayOfWeek)Anterior día de la semana indicado

Ejemplo with() + ajustadores

LocalDate hoy = LocalDate.now();

// Primer día del mes siguiente
LocalDate primero = hoy.with(
    TemporalAdjusters.firstDayOfNextMonth()
);

// Saber qué día de la semana es
DayOfWeek dow = primero.getDayOfWeek();

// Último día del mes actual
LocalDate ultimo = hoy.with(
    TemporalAdjusters.lastDayOfMonth()
);
07 — Diferencia entre fechas y horas

ChronoUnit.between()

LocalDate nacimiento = LocalDate.of(1995, 3, 20);
LocalDate hoy = LocalDate.now();

long anios = ChronoUnit.YEARS.between(nacimiento, hoy);
long dias  = ChronoUnit.DAYS.between(nacimiento, hoy);
long meses = ChronoUnit.MONTHS.between(nacimiento, hoy);

System.out.println("Años: " + anios);
YEARSDiferencia en años
MONTHSDiferencia en meses
DAYSDiferencia en días
HOURSDiferencia en horas
MINUTESDiferencia en minutos
SECONDSDiferencia en segundos

Period — diferencia completa

LocalDate inicio = LocalDate.of(2024, 1, 1);
LocalDate fin    = LocalDate.now();

// Opción 1: until()
Period p1 = inicio.until(fin);

// Opción 2: between()
Period p2 = Period.between(inicio, fin);

System.out.println(p2.getYears());
System.out.println(p2.getMonths());
System.out.println(p2.getDays());
// Ej: "0 años, 3 meses y 7 días"
08 — Parsear fechas (texto → objeto)

parse() — leer desde String

// ISO 8601 por defecto
LocalDate d = LocalDate.parse("2024-07-06");
LocalTime t = LocalTime.parse("08:40:15");
LocalDateTime ldt = LocalDateTime.parse(
    "2024-07-06T08:40:15"
);

// Formato personalizado
DateTimeFormatter fmt =
    DateTimeFormatter.ofPattern("dd/MM/yyyy");
LocalDate d2 = LocalDate.parse("06/07/2024", fmt);

format() — objeto → String

LocalDate hoy = LocalDate.now();

// ISO 8601 (toString por defecto)
System.out.println(hoy);
// → 2024-07-06

// Formato español
DateTimeFormatter esp =
    DateTimeFormatter.ofPattern("dd/MM/yyyy");
System.out.println(hoy.format(esp));
// → 06/07/2024

// Predefinidos de DateTimeFormatter:
// ISO_LOCAL_DATE, ISO_LOCAL_TIME,
// ISO_LOCAL_DATE_TIME, RFC_1123_DATE_TIME...
09 — Formato con idioma (Locale)

DateTimeFormatter + Locale en español

LocalDateTime ldt = LocalDateTime.now();

// Locale español de España
Locale espana = new Locale("es", "ES");

DateTimeFormatter fmt = DateTimeFormatter
    .ofPattern("EEEE, dd 'de' MMMM 'de' yyyy 'a las' HH:mm:ss")
    .withLocale(espana);

System.out.println(ldt.format(fmt));
// → lunes, 06 de julio de 2024 a las 08:40:15

// Usando FormatStyle (formato corto del sistema)
String idioma = System.getProperty("user.language");
String pais   = System.getProperty("user.country");
Locale loc = new Locale(idioma, pais);
DateTimeFormatter auto = DateTimeFormatter
    .ofLocalizedDateTime(FormatStyle.SHORT)
    .withLocale(loc);
System.out.println(ldt.format(auto));
