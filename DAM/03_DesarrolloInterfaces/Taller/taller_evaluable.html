Taller Evaluable - Desarrollo de Interfaces

Enunciado: Implementa la funcion crearTablaHTML que reciba un array de objetos (filas) y un array de strings (columnas). Debe devolver una cadena con el codigo HTML de una tabla (<table>) que muestre los datos. La primera fila debe ser el encabezado con las columnas. Si no hay filas, devuelve una tabla con encabezados y cuerpo vacio.

Ejemplo: crearTablaHTML([{nombre:"Ana",edad:25}], ["nombre","edad"]) debe devolver algo como "<table><tr><th>nombre</th><th>edad</th></tr><tr><td>Ana</td><td>25</td></tr></table>"
