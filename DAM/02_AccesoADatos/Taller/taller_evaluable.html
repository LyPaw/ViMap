Taller Evaluable - Acceso a Datos

Enunciado: Implementa la funcion generarWhere que reciba un objeto con filtros (clave: valor) y devuelva la clausula SQL WHERE correspondiente. Los valores string deben ir entre comillas simples. Si el objeto esta vacio, debe devolver cadena vacia.

Ejemplo: generarWhere({nombre:"Ana",edad:25}) debe devolver "WHERE nombre = 'Ana' AND edad = 25"

Ejemplo: generarWhere({}) debe devolver ""
