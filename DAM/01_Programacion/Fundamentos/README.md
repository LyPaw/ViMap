# Fundamentos de Programacion en Java

## Contenidos

1.  **Sintaxis basica**: estructura de un programa Java, metodo main, paquetes e importaciones.
2.  **Tipos de datos primitivos**: int, double, boolean, char, etc. y sus envoltorios (Wrapper classes).
3.  **Variables y constantes**: declaracion, inicializacion, ambito (scope) y convenciones de nomenclatura.
4.  **Operadores**: aritmeticos, relacionales, logicos, de asignacion y ternario.
5.  **Estructuras de control**: if-else, switch, bucles for, while, do-while.
6.  **Arrays**: declaracion, inicializacion, recorrido, arrays multidimensionales.

## Ejercicios propuestos

1.  Escribe un programa que lea 10 numeros enteros por teclado, los almacene en un array y muestre la media, el maximo y el minimo.
2.  Crea un programa que genere los primeros N numeros de la serie de Fibonacci usando un bucle.
3.  Implementa una calculadora de operaciones basicas usando estructura switch.
4.  Lee una cadena de texto y cuenta cuantas vocales, consonantes, digitos y espacios contiene.
5.  Crea un programa que simule el lanzamiento de un dado N veces y muestre la frecuencia de cada cara.
