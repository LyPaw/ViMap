Taller Evaluable - Programacion

Enunciado: Implementa la funcion filtrarYOrdenar que reciba un array de objetos con estructura { nombre: string, edad: number } y un numero edadMinima. Debe devolver un nuevo array con los nombres de las personas cuya edad sea mayor o igual a edadMinima, ordenados alfabeticamente de forma ascendente.

Ejemplo: filtrarYOrdenar([{nombre:"Carlos",edad:22},{nombre:"Ana",edad:25},{nombre:"Luis",edad:19}], 20) debe devolver ["Ana","Carlos"]
