Taller Evaluable - Mini TODO App con DOM

Enunciado: Implementa un gestor de tareas (TODO) completo con las siguientes funciones. Todas reciben y devuelven datos, sin modificar el DOM directamente (solo manipulan arrays/objetos). Las pruebas unitarias verificaran cada funcion.

  • agregarTarea(tareas, texto) - Agrega una tarea con { id, texto, completa: false }. El id debe ser tareas.length + 1. Devuelve el nuevo array.
  • toggleTarea(tareas, id) - Cambia el estado completa de la tarea con ese id. Devuelve el array actualizado (nuevo array, no mutar original).
  • eliminarTarea(tareas, id) - Elimina la tarea con ese id. Devuelve el nuevo array.
  • filtrarTareas(tareas, filtro) - Devuelve tareas filtradas: "todas" (todas), "activas" (completa===false), "completadas" (completa===true).
  • contarPendientes(tareas) - Devuelve el numero de tareas con completa === false.

No uses mutacion directa: cada funcion debe devolver un NUEVO array u objeto, no modificar el original.
