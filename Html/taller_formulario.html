Taller Evaluable - Validacion de Formularios

Enunciado: Implementa la funcion validarFormulario(datos) que reciba un objeto con campos { nombre: string, email: string, edad: number } y devuelva un array de mensajes de error. Debe validar:

  • nombre: no debe estar vacio
  • email: debe contener "@" y "."
  • edad: debe ser un numero >= 18

Si no hay errores, devuelve array vacio.
