Taller Evaluable - Manipulacion del DOM

Enunciado: Implementa la funcion crearLista(items) que reciba un array de strings y devuelva un elemento <ul> del DOM con un <li> por cada item. Si el array esta vacio, devuelve un <ul> vacio.

Ejemplo: crearLista(["A","B","C"]) debe devolver un <ul> con 3 <li>.

Usa document.createElement y appendChild.
